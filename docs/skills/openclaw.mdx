Caishen skills are compatible with skill-driven agent workflows.

## Recommended pattern

1. Provision wallet runtime via CLI
2. Load skill manifests from `packages/skills/*/SKILL.md`
3. Route write actions through explicit operator confirmation
4. Persist and monitor activity logs continuously
